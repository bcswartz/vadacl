Customer Profile
Demonstration of the customer user profile use case described in the README file. Demonstrates setting validations on a domain class, how to extend the vadacl class with the component class, and how to add and override validations.
Admin Profile
Demonstration of the admin user profile use case described in the README file. Demonstrates the reuse of the domain-level validations in a different component
Patient Form
Demonstrates use of default validation method messages and the withinLength validation method. Also demonstrates validating a sub-FormGroup within the main form FormGroup.
Address Form
Demonstrates setting validations on an object literal within the component, and using the vadacl class as a service.
Company Form
Demonstrates the use of validation messages defined in the locale messages object and associated with the matching class properties. Also illustrates alternate ways of displaying the provided error message or displaying one based on an HTML element tied to the error for that form field.
Company Name Form
Demonstrates the display of validation errors for a control that is programmatically updated.
